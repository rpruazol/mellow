# Getting started

- To build `make build`
- To run `make run`
- To stop `make stop`


the client and server will both need `.env` files in order to run....

## Client
- REACT_APP_BACKEND
- REACT_APP_DOMAIN
- REACT_APP_CLIENTID

## Server
- PORT
- PGUSER
- PGPASSWORD
- PGDATABASE
- PGPORT
- POSTGRES_URL
